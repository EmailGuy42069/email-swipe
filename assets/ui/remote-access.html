Access UI from anywhere

For technical users who want this training UI and settings on a stable URL — like reaching your agent via Telegram. Email Swipe does not host this for you; you run the stack.

Recommended setup

One always-on machine (home server or small VPS) running:

  • Your agent + Telegram bot
  • serve-ui.py + MCP watch-preferences.py
  • Data in ~/.config/email-swipe/

Reach the UI via Tailscale (private) or HTTPS + auth on a VPS.

Options at a glance

OptionGood for
Tailscale / ZeroTierPrivate access to home server — best default for power users
VPS + persistent diskPublic URL; run agent + UI on same box
Cloudflare TunnelHome server without opening firewall ports
ngrokQuick tests; weak for daily use on free tier
Same Wi‑Fi onlyMobile URL from server — no extra setup

Security

Training data is sensitive (senders, subjects, rules). Do not expose an unauthenticated UI to the public internet. Prefer Tailscale or HTTPS with login.

Phone home screen

After you have a stable URL: open in Safari/Chrome → Add to Home Screen. The server must keep running on your host.

Ask your agent

With Explore remote access enabled in Settings → Access, your agent can walk you through Tailscale, VPS, or tunnel setup using the full runbook in references/remote-access-power-user.md.
